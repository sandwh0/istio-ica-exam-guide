# Traffic Management

For traffic management, you need to recognise which resource controls which part of traffic flow.

## For the exam you should be confident with the following

1. Implement sidecars for specific workloads.
   - Be confident using selectors.
2. Configure `VirtualService` and `DestinationRule` resources using weights, subsets, and related routing options.
3. Mirroring requests.
4. Circuit breakers.
5. Injecting fault errors and delays.
6. Retries and timeouts.
7. Implement gateways and virtual services.
8. Service entries and virtual services.
9. Workload entries.
