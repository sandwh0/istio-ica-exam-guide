# Tips and Tricks

- In each question, you will SSH into a node. The SSH command is shown at the start of the question. Remember to exit the node when you're finished.
- You do not need to memorise commands. Learn how to navigate the docs, and how to use the man pages. 
- You get partial marks, so even if you cannot complete the full task, do as much as you can.
- Practice, practice & practice! 
- Useful command for validating a namespace configuration:

    ```bash
    istioctl analyze -n <namespace>
    ```

## Exam Strategy

Start with the easiest questions and complete the ones you are most confident about, then return to the harder ones.

This helps you avoid spending too much time early and rushing straightforward questions at the end.
