# Tips and Tricks

- In each question, you will SSH into a specific node. The SSH command is shown at the start of the question. Remember to exit the node when you finish.
- The relevant documentation links are provided in each question.
- You get partial marks, so even if you cannot complete the full task, do as much as you can.
- Useful command for validating a namespace configuration:

    ```bash
    istioctl analyze -n <namespace>
    ```

## Exam Strategy
Start with the easiest questions first and complete the ones you are most confident about, then return to the harder ones. 
This helps you avoid spending too much time early and rushing straightforward questions at the end.
